FAQ - Questions Fréquentes

Administrateur

  • Comment ajouter un nouvel agent ?
    Rendez-vous dans la section "Gestion des Agents" puis cliquez sur "Ajouter un agent".
  • Comment suivre les KPIs ?
    Utilisez le menu "KPIs" pour accéder aux tableaux de bord détaillés.

Centre d'appel

  • Comment planifier une mission ?
    Depuis "Missions", cliquez sur "Nouvelle mission" et remplissez le formulaire.
  • Où trouver les rapports ?
    Accédez à la section "Rapports" dans la barre latérale.

Donneur d'Ordre

  • Comment consulter les documents ?
    Utilisez le menu "Documents" pour voir ou télécharger vos fichiers.
  • Comment envoyer un feedback ?
    La section "Feedbacks" vous permet de donner votre avis sur chaque mission.

Agent

  • Comment remplir mon rapport journalier ?
    Allez dans "Rapport journalier" et complétez le formulaire en fin de journée.
  • Comment accéder à mes formations ?
    Le menu "Formations" liste toutes vos sessions assignées.

Client final

  • Comment voir l'historique de mes appels ?
    Menu "Historique" dans votre espace personnel.
  • Comment modifier mes informations ?
    Menu "Profil" puis "Modifier mes informations".
